BitTorrent Client

Mulithreaded BitTorrent client that connects via TCP to download and upload from/to multiple peers simultaneously written in Java. Saves progress locally to be resumed upon request. Email for code

BitTorrent Client

  • This project consists of 12 classes that each work together to get a working BitTorrent Client that follows the BitTorrent protocol and utilizes TCP to communicate, verify, and transfer data between a peer.
  • This client supports simultaneous multiple downloads and uploads to different peers via multithreading by storing and keeping track of each piece of the torrent downloaded in a global array of pieces.
  • “Pausing” and resuming download is possible by saving the relevant information locally and then resumed again when the same file is requested for download.
